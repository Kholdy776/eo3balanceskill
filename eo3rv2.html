This is an early demo release for EO3Rv2. It is very much incomplete and contains bugs. It is intended only for testing purposes.

Download the latest version only and apply to a vanilla ROM (filename: 5233 Etrian Odyssey III - The Drowned City (US).nds - however any good dump should work)

To apply, download xdelta UI.

Note for Twilight Menu users: Twilight Menu does not properly bypass the anti-piracy features and will freeze during battles. TWLoader is known to work.
